# FfetchJS

### Tech stack: 
- [x] JavaScript
- [x] Node.js
- [x] React

### How to run: 

```bash
git clone git@github.com:monikakaczan/FfetchJS.git
cd FfetchJS
```

```bash
npm install
npm start
go to http://localhost:3000 to view in the browser
```
